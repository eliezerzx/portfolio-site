# Portfólio Pessoal - Eliezer Carvalho

Este projeto é o meu site de portfólio pessoal, desenvolvido para apresentar minha trajetória acadêmica, experiências, certificações, contatos e alguns dos projetos que venho construindo na área de tecnologia.

O objetivo deste site é reunir, de forma organizada, informações importantes sobre meu perfil profissional, servindo como apoio para recrutadores, networking e divulgação dos meus estudos e projetos.

## Sobre o projeto

O portfólio foi desenvolvido com foco em simplicidade, organização e apresentação clara das informações.

Na página principal, o site apresenta:

- Minha apresentação profissional
- Resumo sobre minha trajetória
- Experiência como Jovem Aprendiz de Redes
- Formação acadêmica
- Certificações e cursos
- Área para acesso ao currículo e certificados
- Informações de contato

Além disso, o projeto possui uma página separada de projetos, onde apresento aplicações desenvolvidas em Python e outros estudos voltados para programação e tecnologia.

## Tecnologias utilizadas

- CSS
